# 🎬 EmojiQuiz — Угадай по эмодзи

Десктопная викторина на C# (WinForms): на экране показываются эмодзи, нужно угадать, что они означают (название фильма). Данные подготовлены на основе открытых датасетов Kaggle и IMDb.

## Возможности

- Случайные вопросы из базы данных SQLite
- Раунд из 10 вопросов (или меньше, если в выбранной категории вопросов не хватает)
- Таймер на ответ (15 секунд)
- Полоса прогресса по вопросам раунда
- Подсветка правильного/неправильного ответа
- Фильтр по категориям
- Экран результата по итогам раунда
- Режим администратора: добавление новых вопросов в базу (с проверкой на пустые поля, дубликаты и корректность ввода эмодзи)
- Все добавленные администратором вопросы сохраняются между запусками

## Стек

- **C# / .NET 10**, WinForms — десктопный интерфейс
- **SQLite** — база данных (файл `quiz.db`, создаётся автоматически при первом запуске)
- **EF Core** (Entity Framework Core) — ORM для работы с базой
- **Python + pandas** — подготовка исходных данных (этап 1)

## Как запустить

### Требования

- Установленный [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- Windows (WinForms работает только на Windows)
- Visual Studio 2022 (актуальная версия, поддерживающая .NET 10)

### Шаги

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Ritka27/EmojiQuiz.git
   cd EmojiQuiz
   ```

2. Откройте файл решения `EmojiQuiz.sln` в Visual Studio 2022.

3. Запустите проект (зелёная кнопка ▶ в Visual Studio, либо `dotnet run` в терминале из папки проекта).

При первом запуске автоматически:
- создаётся файл базы данных `quiz.db`,
- база заполняется вопросами из файла `movies_ru_emoji.tsv`.

## Как пользоваться

**Главное меню** — выбор категории вопросов и переход в игру или в режим администратора.

**Игра** — на экране показывается эмодзи, нужно выбрать один из вариантов ответа. Есть таймер на размышление и счётчик правильных ответов. В конце раунда показывается итоговый результат.

**Администратор** — позволяет добавить новый вопрос (эмодзи, ответ, категория). Новый вопрос сразу появляется в игре и сохраняется в базе данных после перезапуска приложения.

## Подготовка данных (этап 1)

Файл `movies_ru_emoji.tsv` — результат подготовки данных Python-скриптом: исходный датасет [Movies to Emojis](https://www.kaggle.com/datasets/tarundalal/movies-to-emojis-dataset) с Kaggle (английские названия фильмов и эмодзи) объединён с русскими названиями из [официальных датасетов IMDb](https://datasets.imdbws.com/) по идентификатору фильма (`tconst`).
